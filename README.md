# volleyball-stats-racker

  A simple mobile-first volleyball stat tracker for middle school matches.

  ## Features

  - Track our score and opponent score
  - Track serving points
  - Track passing points
  - Track other points
  - Track serve errors
  - Track passing errors
  - Track other errors
  - Assign stats to player jersey numbers
  - View play-by-play log
  - Undo the last action
  - Reset the match
  - Export match stats to CSV
  - Saves the current match in the browser with localStorage

  ## How to Use

  1. Open the app on a phone or computer.
  2. Enter player jersey numbers, like `2, 4, 7, 10, 12`.
  3. Tap `Save`.
  4. Select a player number, or use `No Player`.
  5. Tap the stat button after each play.
  6. Use `Export CSV` after the match.

  ## Notes

  Match data is saved only in the browser on the device being used. It does not
  sync across devices.
